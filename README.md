# java---发送邮件
## 1.传输协议
    SMTP协议（发送邮件）：
    
    我们通常把处理用户smtp请求(邮件发送请求)的服务器称之为SMTP服务器(邮件发送服务器)。
    
    POP3协议（接收邮件）：
   
    我们通常把处理用户pop3请求(邮件接收请求)的服务器称之为POP3服务器(邮件接收服务器)。
## 2.使用java发送邮件

### 1）需要到qq邮箱或者163邮箱开启开启POP3/SMTP服务，并获取授权码
### 2）需要用到外部jar包
    mail.jar   javax.mail

## 3.代码示例
    com.zsc.demo    里面记录的发各种邮件的demo
    com.zsc.mail    封装好了一个发邮件的工具类
